# 🍽 Preparación de la Receta de Arroz con Pollo

## 🥄 Paso 1: Lavar el pollo

Primero se debe lavar bien el pollo para eliminar cualquier suciedad.  
Luego se agrega sal y pimienta al gusto.

```html
<li>Lavar el pollo.</li>
```

---

# 🔥 Paso 2: Freír el pollo

Se coloca aceite en una sartén y se fríe el pollo hasta que quede dorado.

```html
<li>Freír el pollo hasta dorarlo.</li>
```

---

# 🌿 Paso 3: Preparar el culantro

El culantro se lava y luego se licúa con un poco de agua para darle sabor y color al arroz.

```html
<li>Licuar el culantro.</li>
```

---

# 🍚 Paso 4: Cocinar el arroz

Se agrega el arroz junto con el culantro licuado y las verduras como zanahoria y arvejas.

```html
<li>Agregar el arroz y verduras.</li>
```

---

# ⏲ Paso 5: Dejar cocinar

Se deja cocinar aproximadamente durante 20 minutos hasta que el arroz esté listo.

```html
<li>Cocinar por 20 minutos.</li>
```

---

# 🍽 Paso 6: Servir

Finalmente se sirve caliente y puede acompañarse con salsa criolla.

```html
<li>Servir caliente.</li>
```
